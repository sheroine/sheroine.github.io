Documentation

This is a project by Mandy Wong and Joseph Hong.

  1. Our inspiration was from a game called "Just Shapes and Beats" and we feel confident that we tried well to imitate to its layout. The visual looks clean and simple yet there are a lot of movements going on and 3 songs have different layouts so there is quality shown. The UI is not too complex and all of the controls needed for this project is in the GUI on the top right. We followed the instructions in terms of media and code structure as well. However we didn't have radio button groups, because it is not an option in dat.gui. Originally we had radio buttons that let users pick between viewing frequency data or waveform data, we changed it to a dropdown selection insteaad because of dat.gui.
  2. We attempted to make this project into a game, just like the original where the player can move with keyboard and dodge bullets (or anything in solid color) that spawn to the beat of the music. While the bullet spawning worked well, we struggled to finish other gaming elements such as collision detection due to limitations of our time. We replaced keyboard to mouse inputs but collision detection and score is not done. We do know, however, that we will work on this after the submission so it looks more like a game. There is one bug with the player selection, the player keeps defaulting back to Blue when you switch tracks before you start the music. The distortion also doesn't work as of now.
  3. I think not implementing more complex stylish effects to the visualizer and instead investing our time to gamify this project is what didn't go well; even though we had good ideas and plans to make it happen, it didn't happen, for now. What went well, however, is working and communicating together and making something that we want to continue making after submission. Also, the basic requirements were mostly done so checking off our list went well too.
  4. Since we put a lot of our time into this project and even though our promise of making this into a game wasn't fully kept, looking this as a audio visualizer, we think we deserve between 85-90.
