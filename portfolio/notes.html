How I made the site

I made the site with the help of Bootstrap. I used the html tags for modals and carousels to give extra information about my projects. Most of the layout is made with the help of Boostrap as well, but I still put in some extra css to layout the elements better. The colors of the page are all done with css by myself, but I picked the color palette with help from paletton.com

Images

Most images are assets I drew by myself and put together in an image to show off on the site, or screenshots of my own projects. The images in the Geometry Dash Levels are screenshots of the game Geometry Dash, but the screenshots are of the levels I made in the game by myself.

Icons
